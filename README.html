Image Segmentation

This project performs Image Segmentation.

How to get started:

Load the .sln file in visual studio to run the project.
class Pixel Structure:
class Pixel
{
public int PixelRow;
public int PixelCol;

public Pixel
{
PixelRow = 0;
PixelCol = 0;
}

public Pixel(int row,int col)
{
PixelRow = row;
PixelCol = col;

}

}
.

"DoOpenCv" is the function for Image Segmentation it takes eight parametes.
First parameter is the threshold that you want to use for segmentation.
Second and third parameter are the coordinate in the image where you want to start segmentation.
The fourth parameter is the time for example after for how much time you want the changes to remain displayed on the screen.
The Fifth parameter is the label that you will place in the Segmented image to identify the segments.
The sixth parameter is the image that you will use to store result of segmentation.
The seventh parameter is used to state weather sixth parameter should be used or not.
The eight parameter is used to initilize the output matrix of segmented image if we do not wish to use sixth parameter.
